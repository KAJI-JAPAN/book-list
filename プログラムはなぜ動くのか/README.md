# プログラマー脳

## 概要
プログラムが実際にCPU上でどのように動くのかを知ることでコンピューターの解像度を上げ
技術レベルを向上させたい

- コンピューターが小数点の計算を間違える理由
  - 2進数で表せる小数点には限界がある
    - 10進数で10を割り切れないのと一緒3.33333...となる
    - 2新数を10進数に戻したときに同じように誤差が出る
    - 2進数が正確に表せる小数点以下の数字は以下
      - 0.5
      - 0.25
      など他にもある
  - ソースファイルから実行可能ファイルができるまで
    - プログラム言語はネイティブコード(マシン語)に変換しなくては実行できない
      - これを変換してくれる仕組みがコンパイラ
      - 変換することをコンパイルという
      - 例) C言語で作成したプログラミング言語 → Cコンパイラ → ネイティブコード → CPU
    - ビルド
      - ビルドは変換したネイティブコード同士をリンカーでリンクしてCPUで実行可能な状態のファイルにすること
